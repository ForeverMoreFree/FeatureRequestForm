Welcome to our feature request form!

The instructions to use this site are below

Quick start:

  • Click "Add Feature" on navigation bar
    Fill out the needed information
    If chosen priority for this client is taken, others will be shifted accordingly
    Press the "Submit" button

Instructions:

  • Connect/disconnect with database using the "DB Button" on the right of the nav bar
    While disconnected the site will function as it were connected to the server
    • (except priority management)
    On db reconnection, the local database will be overwritteb with the server database

  • To view active feature requests, click "View Features" on the navigation bar
    Click on the appropriate client to view active requests
    You may remove a feature request by pressing the "Complete" button
    Features requests by client are listed in order of priority

  • To edit a feature, click a feature while on the "View Features" page
    You will be redirected to "Update Feature"
    Edit the information as needed
    Press the "Update" button
    You will be redirected to the "View Features" page

  • To remove a feature do one of the following:
    Press "Complete" while on "View Features"
    Press "Delete" while on "Update Feature"
